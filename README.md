## Project setup
```
npm install at root of app
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your unit tests, (Mocha and Chai)
```
npm run test:unit
```

### Run your end-to-end tests (Cypress)
```
npm run test:e2e
```

### Lints and fixes files
```
npm run lint
```
